Key Design Decisions

  • 3 pinned focus projects — you choose which 3 are "right now." Everything else lives in All Projects. Limits overwhelm.
  • One next step per project — the golden arrow. Forces clarity on what actually moves each project forward.
  • Auto-pulled status — "Last commit 2h ago · Deployed ✓" comes from GitHub/Vercel APIs. You never type this.
  • GTD lifecycle badges — Inbox, Someday/Maybe, Exploring, Building, Live, Maintenance
  • Quick Add Idea — brain dump in 5 seconds, goes straight to Inbox. Deal with it later.
  • Dark theme — easier on eyes between patients, uses your brand teal + gold
